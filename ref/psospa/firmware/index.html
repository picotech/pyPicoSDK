Firmware / Updates

Bases: PicoScopeBase, shared_ps6000a_psospa

PicoScope OSP (A) API specific functions

Methods:

Name Description
check_for_update

Query whether a firmware update is available for the device.
start_firmware_update

Begin installing any available firmware update.

check_for_update(n_infos=8)

Query whether a firmware update is available for the device. Args: n_infos: Size of the firmware information buffer. Returns: tuple[list, bool]: (firmware_info, updates_required) where firmware_info is a list of :class:PICO_FIRMWARE_INFO structures and updates_required indicates whether any firmware components require updating.

start_firmware_update(progress=None)

Begin installing any available firmware update. Args: progress: Optional callback (handle, percent) that receives progress updates as the firmware is written.
